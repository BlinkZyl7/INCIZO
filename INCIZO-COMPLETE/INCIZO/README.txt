WHAT IS HYRO AI PRO?
─────────────────────
HYRO AI Pro watches your trading chart in real time and 
gives you AI-powered analysis every few seconds.

It reads your chart like an expert trader would — checking 
RSI, price levels, candle patterns, and momentum — then 
tells you exactly what to do: Long, Short, or Wait.

Built specifically for funded account traders on platforms 
like CLEO (Hydrotrader), TradingView, Binance, and more.

QUICK START (3 STEPS)
──────────────────────

STEP 1 — INSTALL
  Double-click: INSTALL.bat
  This automatically installs everything you need.
  (Requires Python — download free at python.org)

STEP 2 — GET YOUR API KEY (FREE)
  1. Go to: https://console.anthropic.com
  2. Create a free account
  3. Click "API Keys" → "Create Key"
  4. Copy the key (starts with sk-ant-...)
  
  Cost: ~$0.01-0.02 per scan. $5 in credits = 250-500 scans.
  New accounts get free credits to start.

STEP 3 — LAUNCH & CONFIGURE
  1. Double-click: LAUNCH.bat (or run INSTALL.bat again)
  2. Click the SETTINGS tab
  3. Paste your API key
  4. Enter your account size and risk settings
  5. Click SAVE SETTINGS
  6. Go to WATCH tab → click START WATCHING

FEATURES GUIDE
───────────────

📡 WATCH TAB — Live AI Analysis
  • Captures your screen every N seconds automatically
  • AI reads your chart and gives full analysis
  • Shows: Bias, RSI, Entry, SL, TP, Confidence level
  • Sound alert when a new signal is detected
  • Popup notification for Long/Short signals
  • Countdown timer to next scan
  • Live screen preview
  • Manual snap button for instant analysis

📓 JOURNAL TAB — Trade Tracking
  • Log every trade with one click
  • Auto-calculates P&L from entry/exit prices
  • Tracks win rate, total P&L, today's P&L
  • Challenge progress bar showing % to target
  • Export all trades to CSV for tax/review
  • Shows last 20 trades with color coding

🧮 RISK TAB — Position Size Calculator
  • Enter account size, risk %, entry, SL, TP
  • Instantly calculates exact position size
  • Shows risk amount in dollars
  • Calculates R:R ratio with warnings
  • Tells you if R:R is good or bad for funded accounts

⚙ SETTINGS TAB
  • API key management (saved securely)
  • Account size and risk % configuration  
  • Profit target tracking
  • Toggle sound/popup alerts on or off
  • All settings auto-saved between sessions

TIPS FOR BEST RESULTS
──────────────────────
✓ Keep your chart FULLSCREEN when using the watcher
✓ Use 5M or 15M timeframe for scalping signals  
✓ Set scan interval to 10-15 seconds (not too fast)
✓ Always confirm the signal on your chart before trading
✓ Never risk more than 1-2% per trade on funded accounts
✓ Log every trade in the Journal — discipline builds profit

FOR FUNDED ACCOUNT TRADERS:
✓ Set your account size to match your funded account
✓ Set risk % to match your platform's rules (usually 1%)
✓ Use the Risk Calculator before every single trade
✓ Watch your daily loss limit in the Journal tab

TROUBLESHOOTING
────────────────

"Python not found" error:
  → Download Python from python.org
  → During install, CHECK "Add Python to PATH"
  → Restart your computer, then run INSTALL.bat again

"API Error" in the app:
  → Check your API key is pasted correctly in Settings
  → Make sure you have credits at console.anthropic.com
  → Check your internet connection

App won't open:
  → Right-click INSTALL.bat → "Run as Administrator"
  → Try running: python hyro_ai_pro.py in Command Prompt

Analysis says "can't see chart":
  → Make sure your chart is visible on screen (not minimized)
  → Keep CLEO/TradingView as the main window
  → Try the 📷 SNAP button for a manual scan
